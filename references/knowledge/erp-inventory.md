# ERP/进销存估算知识库

## 适用范围

适用于采购、销售、库存、仓库、供应商、客户、单据流等进销存或轻量 ERP 系统。不适用于大型制造 MRP、复杂财务总账或多组织集团 ERP，除非单独拆分。

## 常见一级模块

- 采购
- 销售
- 库存
- 仓库
- 供应商
- 客户
- 单据流
- 报表

## 模块估算基准

### 采购管理

常见二级功能：
- 采购申请
- 采购订单
- 采购入库
- 采购退货
- 供应商价格

默认复杂度：
- 单据增删改查：M
- 审批流和多状态流转：H
- 与库存/财务联动：H

基准工时：
- 产品：1-2
- Web后台：2-4
- 后端：3-6
- 测试：1.5-3

客户确认问题：
- 单据是否需要审批？
- 采购入库是否直接影响库存？
- 是否需要供应商价格历史？

### 库存管理

常见二级功能：
- 库存查询
- 入库
- 出库
- 调拨
- 盘点
- 库存预警

默认复杂度：
- 单仓库存：M
- 多仓、多批次、序列号：H
- 库存成本核算：H

基准工时：
- 产品：1-2
- Web后台：2-5
- 后端：4-8
- 测试：2-4

常见风险：
- 库存扣减时机不明确
- 批次、效期、序列号会显著增加复杂度

客户确认问题：
- 是否支持多仓库？
- 是否管理批次、效期或序列号？
- 库存是否允许负库存？

### 销售管理

常见二级功能：
- 销售订单
- 发货
- 退货
- 客户价格
- 应收关联

默认复杂度：
- 普通销售单：M
- 价格政策和应收联动：H

基准工时：
- 产品：1-2
- Web后台：2-4
- 后端：3-7
- 测试：1.5-3

客户确认问题：
- 是否有客户等级价格或合同价？
- 销售出库是否需要审批？
